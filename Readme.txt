In this project: Music Player

Introduction:

In this project when i(User) play 3 or 4 songs (or change the 3 or 4 songs) give the message (popup) that "please pay to play the song" if i(User) click on "payment" button then continue the song.
 
else play only first song until User will pay

File Structure:- 
./index.html    ===> This is the HTML file
./code/app.js   ===> This is the JavaScript file for backend
./index/data.js ===> This is the JavaScript file to access the media (define array or object)
./media         ===> This is the directory (folder) for all the media



To run the project:- just right click on the "index.html" and click on open in browser (like Edge, Chrome, etc)
